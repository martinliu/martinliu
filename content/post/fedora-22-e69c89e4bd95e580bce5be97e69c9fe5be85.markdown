我使用这新安装的 F21，连接着公司阿姆斯特丹的 VPN，流畅的使用 Firefox ， Chrome 上网，在没有墙的世界里顺利传行着。重返 Linux 世界，想起来很久没有用 windows 桌面了，渐渐适应和熟悉这感觉，且有些怀旧。

之前 Fedora 是没有太多的体验，用的最多的是 opensuse + kde 桌面的组合。目前，看到 fedora magazine 上很多更新，感觉真的是发展的太快了。 http://fedoramagazine.org/see-whats-coming-in-fedora-22-workstation/ 看了下这个帖子，上面说 F22 的提升包括：更长的电池寿命、gnome Wayland 更多改进、开始应用 bundle 的开发、更好的第三方应用支持、提升 Terminal、改善开发工具等等。查看更多细节：[Christian’s blog pos](http://blogs.gnome.org/uraeus/2015/01/19/planning-for-fedora-workstation-22/)t
